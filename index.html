ABOUT EVENT

本大会は、全国のVRに興味を持つ学生が集い、互いに活動や知見を報告するイベントです。VRにワクワクする学生同士が交流することで、全国規模でのVRの発展を目指しています。

参加について

登壇者の募集は終了しました。多数のご応募ありがとうございました。
聴講のみの場合は参加フォームの提出は不要です。開催日時に各会場へ直接お越しください。

イベント概要

  • 内容: 各団体の活動の様子や制作物の紹介、困っていることなどの共有（一団体5分〜最大10分、質疑応答あり）
  • 目的: サークル活動を通した知見・悩み・情熱の交換、モチベーション向上、学生同士の交流
  • 日時・場所:
    - 2025年10月18日（土）19:00開始 @ cluster (18:50開場 / 登壇者18:00集合)
    - 2025年10月19日（日）19:00開始 @ VRChat (18:50開場 / 登壇者18:30集合)
  • 登壇資格: ガイドラインの規定により本大会には高校～大学の学生団体が登壇できます。

聴講方法

聴講はどなたでも可能で、事前の申し込みは不要です。clusterとVRChatには定員がございますので、超えた場合はYouTube配信をご覧ください。

  • 10/18(土) cluster: こちらのclusterイベントページよりご参加ください。
  • 10/19(日) VRChat: 全国学生VRサークル活動報告大会GROUPのGroupインスタンスにて開催します。
  • 両日 YouTube: YouTubeにて生配信（アーカイブあり）を行います。

  運営: UT-virtual, NUMA

制作したXRコンテンツについてご紹介いただくセッションです。優秀な発表には、展示会出展権やQuest3sなど豪華副賞が与えられます。

イベント概要

  • 内容: 制作したXRコンテンツについて、1分程度のプレゼンを実施。優秀発表には副賞あり。
  • 日時・場所:
    - 2025年10月18日（土）15:00開始 @ cluster (14:50開場 / 登壇者14:00集合)
    - 2025年10月19日（日）15:00開始 @ VRChat (14:50開場 / 登壇者14:30集合)
  • 登壇資格: 高校生～大学生（院生含む）。個人での参加も可能です。
  • コンテンツ要件: XRに関連する自作コンテンツであること。詳細は参加フォームをご確認ください。

表彰・副賞について

東京大学バーチャルリアリティ教育研究センター様のご共催により、優秀な発表は表彰され、以下の副賞が与えられます。

  • 東大VRセンター主催の展示会「メタバースコンペティション」への出展権（旅費応相談）
  • MetaQuest3sの贈呈
